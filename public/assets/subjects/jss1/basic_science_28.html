Pole Vault Skills

Pole Vault is a field event under jumps just as the name, is long flexible pole used as an aid to clear a high crossbar without knocking it down. It is an exciting sport event that promotes courage and adventure in the athlete.

 
Basic Skills and Pole Vault Skills Techniques.

The basic skills in pole vault include the following.

  • Grip
  • Run-up/approach
  • Swing -up Pole-up
  • Pull-up and bar clearance
  • Take off.
  • Hang
  • Landing and Recovery
  • Swing up
  • Run up
  • Flyaway

 

Basic Javelin Skills
Javelin Throw

Javelin throw is one of the throwing events in athletics. Other throwing events include; discus, shot put and hammer throw.

 

Basic skills include
  • The grip/hold
  • The carry
  • Approach/run-up
  • Delivery/Release
  • Recovery/Reverse
